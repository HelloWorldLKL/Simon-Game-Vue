# Simon-Game-Vue

> 这是一款使用 Vue.js 制作的 Simon Game

## 编译

``` bash
# 安装依赖
npm install

# 服务器搭建在 localhost:8080
npm run dev

# 生成生产环境经过压缩的代码
npm run build
```

## 特点

+ 生为练习产物，并没有实用价值
+ 重于 Vue.js 的联系与逻辑的处理，目前较为熟悉了 Vue.js 的 *自定义事件* 和 *内容分发*


+ 使用 Vue.js 框架处理逻辑，方便后续的开发
